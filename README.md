# SmartPhone Shop
It is a web application where people people can purchase smartphones online

## Built With

- .NET 5
- ASP.NET Core 5.0
- Docker
- MongoDB
- PostgreSQL
- SQL Server
- RabbitMQ
- MassTransit
- Ocelot API Gateway
- AutoMapper
- CQRS
- Dapper
